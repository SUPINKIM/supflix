# Supflix

Show the Movies and TV shows Lists on the beautiful website ✨

## Preview:

Try it on [github-pages](https://supinkim.github.io/supflix/)

## Screens

- [✔︎] Home
- [✔︎] TV Shows
- [✔︎] Search
- [✔︎] Detail

## API verbs

- Movies

  - [✔︎] Now Playing
  - [✔︎] Top Rated
  - [✔︎] Popular
  - [✔︎] Upcoming
  - [✔︎] Search
  - [✔︎] Movies Detail

- TVs

  - [✔︎] Top Rated
  - [✔︎] Popular
  - [✔︎] Airing Today
  - [✔︎] Search
  - [✔︎] TVs Detail
